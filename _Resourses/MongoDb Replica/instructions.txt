Convert MongoDb standalone server to Replication

1- Terminal
a) sudo systemctl stop mongod
b) sudo systemctl status mongod

2- Terminal
a) /usr/bin/mongod --replSet "rs0" --config /etc/mongod.conf --fork
b) sudo systemctl enable mongod //TODO check it

3) Mongosh =>
rs.initiate({ _id: "rs0", members: [ { _id: 0, host: "localhost:27017" } ] })
